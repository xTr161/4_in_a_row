Rules:

Connect Four (also known as Four Up, Plot Four, Find Four, Captain's Mistress, Four in a Row, Drop Four, and Gravitrips in the Soviet Union) is a two-player connection board game, in which the players choose a color and then take turns dropping colored discs into a seven-column, six-row vertically suspended grid. The pieces fall straight down, occupying the lowest available space within the column. The objective of the game is to be the first to form a horizontal, vertical, or diagonal line of four of one's own discs.

  • Turn based
  • 2 players
  • form a horizontal line of 4
  • form a diagonal line of 4
  • form a vertical line of 4
  • colors are red and yellow
  • if the board is full with no match of 4 then draw
  • grid of 6x7
  • optional sizes: 5×4, 6×5, 8×7, 9×7, 10×7, 8×8
  • 42 total pieces
  • 21 moves available for each player

Building the game

Backend

  1. Since the game uses rows and columns, we will need to make use of a matrix, which is an array nested in an array.
  2. Player should have the option to select the row and column size to be able to play multiple variations of the game
  3. The game is played by only 2 players but each player has its own attributes as its value and color will be different ,and we need to allow for the possibility of a non-human player
  4. players should be able to choose between playing another human player or playing the computer
  5. The game is turn based thus we will introduce a while loop that checks if there's a winner on every iteration
  6. Define winning conditions
  7. players can only choose columns
  8. Since players can only choose columns but the matrix takes a column and a row value, we need a generator that checks for next available row and thus the row is calculated automatically
  9. Define 3 levels of AI
  10. AI level 1 uses random function
  11. build a scoring system for the minimax algorithm
  12. Build minimax algorithm
  13. Build a medium level using restricted depth
  14. Build an expert level using deep depth
